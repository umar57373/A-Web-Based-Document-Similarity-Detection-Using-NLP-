A Web-Based Document Similarity Detection Using NLP
Project Info
A web application that detects similarity between documents using Natural Language Processing (NLP) techniques.
Technologies Used

Vite
TypeScript
React
shadcn-ui
Tailwind CSS
Python (FastAPI backend)
NLP / Cosine Similarity

How to Run Locally
Frontend
# Step 1: Clone the repository
git clone https://github.com/umar57373/A-Web-Based-Document-Similarity-Detection-Using-NLP-.git

# Step 2: Navigate to the project directory
cd A-Web-Based-Document-Similarity-Detection-Using-NLP-

# Step 3: Install dependencies
npm install

# Step 4: Start the development server
npm run dev
Backend
cd backend
pip install -r requirements.txt
python similarity_api.py

How to Deploy
Then host the dist/ folder on Vercel, Netlify, or any static hosting service.
